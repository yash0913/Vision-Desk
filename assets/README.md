# Assets Directory

This directory contains icons and images for the DeskLink application.

## Required Icons

You need to add the following icon files:

1. **icon.png** - Application icon for system tray (256x256 PNG)
2. **installer-icon.ico** - Installer icon (32x32 ICO)
3. **uninstaller-icon.ico** - Uninstaller icon (32x32 ICO)  
4. **header-icon.ico** - Installer header icon (64x64 ICO)

## Icon Guidelines

- Use a consistent design theme
- Ensure good visibility at small sizes
- Include transparency for PNG files
- Test on different Windows themes

## Temporary Solution

For development/testing, you can:
1. Download free icons from sites like Flaticon or Icons8
2. Use online converters to create ICO files from PNG
3. Use the Windows icon converter tools

## Note

The application will still work without these icons, but will use default Electron/system icons instead.
